# üî• LoRA Forge

> A streamlined web interface for configuring, training, and managing LoRa (Low-Rank Adaptation) fine-tuning experiments on Large Language Models, built on Google Cloud Vertex AI.

This tool provides a simple UI to abstract away the complexity of setting up training jobs, allowing you to focus on experimenting with models and datasets.

## üöÄ Features

* **Intuitive UI:** Easily configure all key training parameters (Model, Dataset, Epochs, Learning Rate, LoRa Rank, etc.).
* **AI Assistant:** An integrated AI assistant to help guide your configuration.
* **Live Training Logs:** Monitor the progress of your training jobs in real-time.
* **One-Click Start:** Launch and manage training runs with simple button controls.
